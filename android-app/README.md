# Чистое нативное Android приложение (Kotlin)

Вы абсолютно правы: React Native, хоть и компилируется в приложение, всё равно использует JavaScript и веб-технологии под капотом.

Если вам нужен **истинно мобильный код**, который написан специально под Android и работает без всяких браузерных ограничений — это язык **Kotlin** (официальный язык разработки под Android от Google).

В этой папке `android-app` я создал для вас структуру и исходный код **чистого Android-приложения**.

## Главный плюс этого приложения:
Оно **вообще не знает, что такое браузерные блокировки (Mixed Content)**. 
Вам больше не нужны никакие Ngrok или туннели! Вы просто вводите локальный IP компьютера (например, `192.168.1.55`), и приложение моментально подключается к нему по домашнему Wi-Fi через обычный `ws://`.

## Как это запустить и скомпилировать в .apk?

Поскольку это настоящий Android-код, его нельзя запустить через браузер или Node.js. Вам понадобится официальная среда разработки:

1. Скачайте и установите **Android Studio** на ваш ПК (это бесплатно).
2. Скачайте эту папку `android-app` к себе на компьютер.
3. Откройте Android Studio и нажмите **Open** -> выберите скачанную папку `android-app`.
4. Подождите пару минут, пока Android Studio скачает нужные библиотеки (Gradle sync).
5. Подключите свой телефон к ПК по USB (включив на нем "Отладку по USB" в настройках разработчика).
6. Нажмите зеленую кнопку **Play (Run)** вверху экрана.

Android Studio сама скомпилирует настоящий `.apk` файл, установит его на ваш телефон и запустит!

### Что внутри кода?
* `MainActivity.kt` — Вся логика. Написана на Kotlin с использованием современного UI-фреймворка **Jetpack Compose**. Для связи с сервером используется мощная библиотека **OkHttp**.
* `AndroidManifest.xml` — Файл настроек, где мы явно просим у Android разрешение на доступ к локальной сети (`INTERNET`) и разрешаем подключаться без шифрования (`usesCleartextTraffic="true"`).
