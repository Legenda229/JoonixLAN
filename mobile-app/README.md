# PC Remote Native App

Это полноценное мобильное приложение (React Native / Expo), которое компилируется в настоящий `.apk` для Android или `.ipa` для iOS. Оно не использует браузер и работает как нативное приложение.

## Как запустить на своем телефоне?

1. **Установите Node.js** на ваш компьютер (если еще не установлен).
2. Скачайте папку `mobile-app` к себе на компьютер.
3. Откройте командную строку в этой папке и выполните:
   ```bash
   npm install
   ```
4. Запустите сервер разработки Expo:
   ```bash
   npx expo start
   ```
5. **На телефон:** Скачайте приложение **Expo Go** (из Google Play или App Store).
6. Откройте Expo Go и отсканируйте QR-код, который появился в командной строке на ПК.

Приложение моментально откроется на вашем телефоне!

## Как скомпилировать в .apk (установочный файл для Android)?

Чтобы получить настоящий файл `.apk`, который можно скинуть друзьям или установить навсегда:

1. Установите утилиту EAS CLI:
   ```bash
   npm install -g eas-cli
   ```
2. Авторизуйтесь (потребуется аккаунт на expo.dev):
   ```bash
   eas login
   ```
3. Запустите сборку:
   ```bash
   eas build -p android --profile preview
   ```
4. Подождите 10-15 минут, и в консоли появится ссылка на скачивание вашего готового `.apk` файла!
